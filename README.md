## 🎯 Project Overview

**AILINK** is a decentralized AI-Web3 interaction network built on **BNB Chain**.  
Users earn `ALINK` tokens through micro-games, AI media creation, NFT minting, and community participation.

### Core Pillars
| Pillar | Description |
|---|---|
| 🎮 **Micro-Games** | Daily games, missions, leaderboard rewards |
| 🤖 **AI Media Suite** | AI image, video, music generation |
| 💎 **NFT Minting** | Mint AI creations as BEP-20 NFTs |
| 🏛️ **DAO Governance** | Token holders vote on platform direction (2026) |

## 📁 Project Structure

```
ailink-web/
├── src/
│   ├── index.tsx              # Hono app — 전체 라우트 (10개)
│   └── pages/
│       ├── layout.ts          # 전역 HTML shell (navbar, footer, CSS, auth)
│       ├── home.ts            # 홈페이지 (14섹션, ~700줄)
│       ├── login.ts           # 로그인 (지갑/이메일, localStorage 인증)
│       ├── signup.ts          # 회원가입
│       ├── mypage.ts          # 마이 대시보드 (KPI, 활동, 베스팅)
│       ├── vesting.ts         # 베스팅 상세 (차트, 월별 테이블)
│       ├── whitepaper.ts      # 백서 v1.1
│       └── legal.ts           # Privacy / Terms / Disclaimer
├── public/
│   └── static/                # 정적 에셋
├── dist/                      # 빌드 결과물 → Cloudflare Pages
├── ecosystem.config.cjs       # PM2 (샌드박스 개발용)
├── wrangler.jsonc             # Cloudflare 설정
├── vite.config.ts             # Vite 빌드 설정
└── package.json
```

## 🚀 Development

```bash
# 설치
git clone https://github.com/vinsenzo83/ailink-web.git
cd ailink-web
npm install

# 로컬 빌드 & 실행 (샌드박스)
npm run build
pm2 start ecosystem.config.cjs
# → http://localhost:3000
```
